## 2.MarkDown 在 Mac 上面预览<a name="preview"/>
* 1.默认 Mac 上面的 `*.md` 文件是没有预览功能的，所以就需要安装一个预览的插件: [qlmarkdown](QuickLook generator for Markdown files)
* 2.相关的命令:

```
#To install
$ brew update

$ brew cask install qlmarkdown

#To uninstall:

$ brew cask uninstall qlmarkdown
```
